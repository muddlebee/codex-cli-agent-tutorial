# m4 - Approvals, Steer, and Interrupt

## What was added
- Interactive approval handling in RPC mode (`approve action? [y/N]`)
- `nano-agent steer <sessionId> "<text>"`
- `nano-agent interrupt <sessionId>`
- RPC provider support for `turn/steer`, `turn/interrupt`, and approval responses

## Why this matters
This milestone turns the CLI into an active controller over in-flight Codex turns instead of only a passive event reader.
