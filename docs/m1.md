# m1 - Interactive Chat and Persistence

## What was added
- `nano-agent chat` REPL mode
- Persistent session metadata and transcript in `.nano-agent/sessions/<id>/`
- Resume support: `nano-agent chat --resume <id>`
- Session listing: `nano-agent sessions`

## Why this matters
You now have a usable CLI workflow with continuity across runs, while still using the Codex exec control plane.
